Detailed Description

FIXED to allow element-by-element namespacing. Current built-in implementatin builds XML without reference to a namespace until the end, and PHP DomDocument will strip the namespace from elements before the namespace is introduced.

This class is the CakePHP-supplied class with changes introduced. To use it, don't replace the Cake file. Override it in your AppController by adding it to your $helpers array like this: 

      public $helpers = array( 'Rss' => array('className' => 'FixedRss'));

Now your application will use the new class instead of the old.

To use define your |item| as an array, with an array 'namespace' containing a string or or an array with keys prefix and url., e.g.,

$this->Rss->item(array(), array( 'content:encoded' => array( $myContent, 'n$post['BlogPost']['post_title'], );
